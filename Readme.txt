1、>>testing
	新建5*5的机器人，整体名为 robot ，彼此之间没有连接
	每个模块分别命名为a1/a2/a3...b1/b2....e4/e5

2、连接/断开模块，函数为
   >>connectcube(a1,a2) or disconnectcube(a1,a2)
	a1,a2可为任意两个模块机器人，如果之间距离太大太小会显示无法连接
	没有写重载函数，只能两个两个连接。。。

3、旋转机器人，函数为
   >>robotrotate(robot,a1,a2,90)
	robot是整个机器人的名字，a1,a2为发生转动的两个模块，转动轴为a1,a2中心连心线，方向是a1指向a2，90为度数，从a2看向a1逆时针为正。
   >>robotplot(robot)
	显示现在的机器人的形状

   如果要旋转并且看到动画，函数为
    >>robotrotateplot(robot,a1,a2,90)
   但是很卡，非常的卡顿，和PPT一样，还在找问题原因。

2、3过程交替就可以完成模块化机器人的运动
